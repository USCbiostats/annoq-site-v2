# AnnoQ Site V2 Repository Guide

This folder documents the React port of the AnnoQ website. It is meant for a developer who has never seen the project before and needs to understand what the site does, where the code lives, and how the main flows work.

## What This Site Is

AnnoQ is a web interface for querying large-scale genetic variant annotations. Users can:

- Read static project information on the landing, about, release/news, contact, cookie policy, and docs pages.
- Browse supported annotation fields from the backend annotation tree.
- Select annotation fields and export/import annotation configs.
- Submit variant queries by chromosome range, VCF rows, gene product, rsID, or rsID list.
- View paginated results in a wide table.
- Filter results by annotations that have values.
- Open row details, column summaries, and column-level stats.
- Download result files generated by the backend.

The React app preserves the professional Material-style feel of the old Angular site, but the implementation is intentionally cleaner: query state is explicit, API calls are cancellable, annotation selection is shared through context/localStorage, and the result table avoids the old rendering freezes.

## Quick Start

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build production assets:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Important Entry Points

- App bootstrap: `src/main.tsx`
- Routing shell and top nav: `src/App.tsx`
- Global theme: `src/theme.ts`
- Global CSS: `src/styles.css`
- Static pages: `src/pages/StaticPages.tsx`
- Docs markdown renderer: `src/pages/DocsPage.tsx`
- Supported annotations page: `src/pages/SupportedAnnotationsPage.tsx`
- Query UI shell: `src/features/search/SearchWorkspace.tsx`
- Query form drawer: `src/features/search/QueryDrawer.tsx`
- Results table: `src/features/search/ResultsTable.tsx`
- Query reducer/store: `src/features/search/searchState.tsx`
- Backend requests: `src/lib/api.ts`
- GraphQL query builders: `src/lib/queryBuilder.ts`
- Annotation tree/store helpers: `src/lib/annotations.ts`
